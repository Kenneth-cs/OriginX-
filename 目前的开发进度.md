# 源极科技 (OriginX) 官网 - 当前开发进度报告

## 🛠️ 技术栈与架构
- **核心框架**：React 18 + React Router 7 + Vite
- **样式与UI**：Tailwind CSS + shadcn/ui (基于 Radix UI)
- **动画引擎**：Framer Motion
- **图标库**：Lucide React

## 📊 已完成模块
官网的整体骨架和核心页面已经搭建完毕，视觉上采用了极具科技感和未来感的暗色调（Dark Mode）、毛玻璃效果（Glassmorphism）和流光背景。

### 1. 全局布局 (`Layout.tsx`)
- 带有动态呼吸光晕背景、固定在顶部的毛玻璃导航栏，以及基础的底部 Footer。
- 实现了平滑滚动（Smooth Scrolling）锚点导航逻辑。

### 2. 首页 (`Home.tsx`)
- **Hero 愿景区**：展示震撼的标语和引导探索按钮。
- **业务矩阵 (The Matrix)**：极智・工业、源生・人文、源效・商业三大板块的精美卡片展示。
- **源极实验室 (The Lab)**：展示前沿 AI 探索和 Demo。
- **关于源极 (The Identity)**：精简版品牌介绍，引导跳转至完整品牌页。
- **联络中心 (Terminal)**：设计了开发者/企业双 Tab 切换的“终端”风格的合作指引，并附带了邮箱和微信联系方式。

### 3. 品牌文化页 (`Brand.tsx`)
- 详细阐述了“始于本源，触达极境”的理念。
- 展示了 Logo 设计灵感（中心星核、交错轨道、隐晦泛光）和深空蓝/天际青的色彩美学。

### 4. 源极视界/新闻页 (`News.tsx`)
- 前端界面已完成，实现了新闻列表展示以及点击后查看文章详情的交互。
- *(注：目前数据是静态模拟数据)*
