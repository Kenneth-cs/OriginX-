# 源极科技 (OriginX) 官网 - 管理后台使用说明

本指南旨在帮助非技术人员（运营、市场、商务）快速掌握源极科技官网后台的使用方法。通过本后台，您可以零代码修改官网文案、发布新闻洞察以及处理客户留资。

---

## 二、 客户需求表单 (`/admin/contacts`)

此模块用于接收并处理客户在官网首页底部“合作联系”终端中提交的合作意向。

1. **线索查看**：列表中会按时间倒序展示客户的姓名、公司、联系方式及具体需求。
2. **状态管理**：新提交的线索默认显示为红色的“未读”状态。
3. **处理跟进**：在您或商务团队跟进完毕后，点击右侧的 **“标为已读”** 按钮，将其归档为绿色状态，方便区分新老线索。

---

## 三、 资讯内容管理 (`/admin/news`)

此模块用于管理官网“源极视界”页面的所有文章。

### 1. 发布新文章
- 点击右上角的 **“新建文章”** 按钮。
- **基本信息**：
  - **文章标题**：吸引人的核心标题。
  - **分类**：可选择“行业洞察”、“产品动态”、“实验室探索”等。
  - **列表摘要**：不超过 100 字的简短介绍，会显示在官网的新闻列表卡片上。
- **正文编辑 (Markdown)**：
  - 正文采用 **Markdown 格式** 进行编写。如果您不熟悉，可以把它当成一个支持快捷键和简单标记的文本框。
  - **插入图片**：点击编辑器上方的 **“插入图片”** 蓝色小按钮，选择本地图片，系统会自动上传并生成一段 `![图片](URL)` 的代码。请勿删除这段代码，它在官网上会自动渲染为图片。
- **发布与草稿**：
  - 点击右上角的 **“状态”按钮** 可以在“草稿”与“已发布”之间来回切换。只有状态为“已发布”的文章，普通用户才能在官网上看到。
  - 点击蓝色的 **“保存” / “保存更新”** 按钮完成发布。

### 2. 编辑与删除
- 在文章列表中，找到目标文章，点击右侧的 **“编辑”** 即可修改内容。
- 点击 **“删除”** 会将文章永久移除，请谨慎操作。

## 四、 全站页面配置 (`/admin/settings`)

此模块是整个网站的“控制大脑”。您可以直接在这里修改首页标语、联系方式以及全局 SEO 设置。**修改并点击保存后，官网立即生效，无需程序员重新打包发版！**

**操作说明**：
配置区的数据格式为 **JSON (JavaScript Object Notation)**。修改时请**务必注意以下几点**，否则会导致保存失败或页面报错：
1. **只能修改双引号 `""` 里面的文字内容**。
2. **千万不要**删除大括号 `{}`、中括号 `[]`、英文逗号 `,` 和原有的英文双引号。

### 1. `global_seo` (全局 SEO 配置)
控制网站在浏览器标签页上的标题，以及分享给别人时的摘要描述。
```json
{
  "title": "在这里修改网站标题，如：源极科技 | AI解决方案专家",
  "description": "在这里修改网站介绍...",
  "keywords": ["AI", "源极科技"] 
}
```

### 2. `home_hero` (首页顶部区域)
控制用户打开官网第一眼看到的那些核心震撼标语。
- `"tag"`: 最顶部的小发光标签。
- `"title.line1_highlight"`: 大标题第一行高亮的字（蓝色的“源”）。
- `"title.line1_white"`: 大标题第一行白色的字（“于智能”）。
- `"description"`: 大标题下方的长段描述文案。
- `"quote"`: 段落最后那句金句引言。

> **💡 进阶提示**：如果未来官网开发了新的模块，程序员会在此页面增加新的配置块（如 `contact_info`），您只需按照同样的规则修改文字即可。

## 五、 常见问题与安全须知

1. **退出登录**：操作完毕后，请点击左侧边栏最下方的 **“退出登录”**，以防他人篡改数据。
2. **账号安全**：如有离职交接或安全需要，可联系技术人员修改服务端密码。
3. **图片上传失败**：请检查图片大小，目前系统限制单张图片最大不可超过 **5MB**。
4. **JSON 报错**：如果在“全站页面配置”输入时出现红色错误提示，通常是因为您不小心删除了引号或逗号。请撤销更改（Ctrl+Z / Cmd+Z）或恢复为您需要修改之前的文字。
